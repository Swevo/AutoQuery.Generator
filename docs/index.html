AutoQuery.Generator

Compile-time query composition for IQueryable<T> using Roslyn incremental source generators.

NuGet · GitHub · README

What it generates

[QuerySpec(typeof(Product))]
public partial class ProductQuery
{
    public string? Name { get; set; }
    public decimal? MinPrice { get; set; }
    [QuerySort] public string? SortBy { get; set; }
    public bool SortDescending { get; set; }
    public int PageNumber { get; set; } = 1;
    public int PageSize { get; set; } = 20;
}
var result = new ProductQuery { Name = "Laptop" }.Apply(products);

Zero reflection

No runtime scanning. Everything is emitted at build time.

Composable LINQ

Generated methods return IQueryable<T> so they chain naturally.

Convention + override

Use nullable properties by convention or override with [QueryFilter].
